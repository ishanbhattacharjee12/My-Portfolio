## ✨ Features
- **Dark Theme with Glassmorphism**: Premium aesthetic with blur backdrops and subtle neon gradients.
- **Fully Responsive**: Optimized for desktop, tablet, and mobile views using Flexbox and CSS Grid.
- **Smooth Animations**: Intersect Observer-based scroll reveals and hover micro-interactions.
- **Clean Architecture**: Built purely with Vanilla HTML, CSS, and JS for maximum performance and zero build-step deployments.
- **AI-Focused Content**: Highlights skills in PyTorch, OpenAI API, RAG, FastAPI, React.js, Vector Search, and more.

## 🛠️ Featured AI Projects Showcase
1. **Verity: Enterprise RAG Pipeline**: A production-ready RAG system utilizing hybrid retrieval, LLM reranking, and citation verification. Deployed on Vercel and Render.
2. **AI YouTube Summarizer**: A full-stack application generating structured summaries from YouTube transcripts with timestamp navigation.
3. **Next Word Predictor (LSTM)**: A PyTorch-based neural network for predictive text modeling with 96% validation accuracy.

## 🚀 Deployment
This portfolio is statically hosted on **GitHub Pages**. 
Any changes pushed to the `main` branch are automatically built and deployed.

### Running Locally
To test the site locally, you don't need any special build tools:
1. Clone the repository: `git clone https://github.com/ishanbhattacharjee12/My-Portfolio.git`
2. Open `index.html` in your favorite browser.
   - *Alternatively, use an extension like VS Code Live Server for hot reloading.*
